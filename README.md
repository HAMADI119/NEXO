# NEXO
It is composed of two major parts:  * backend: a backend app (a very simple JSON API built using Node and ExpressJS) * frontend: a single-page frontend app (built using React and create-react-app)
Create a marketplace app called NEXO. 
The app allows merchants to rent digital shop frames for one year using their Pi account. 
Each merchant uses their own Pi identity to manage their shop. 
The app owner only provides the platform (frames) and does not sell products.
Include:
- Home page
- List of shops
- Shop details page
- Merchant profile using Pi login
- Simple and clean UINEXO is a demo marketplace app where merchants rent digital shop frames yearly using their Pi account to display products and reach customers.
